# Проект 1: Столбиковые диаграммы — анализ финансовой эффективности

## Данные
Financial Sample (Microsoft, открытые данные). 700+ строк.

## Задача
С помощью столбиковых диаграмм в Power BI ответить на 3 бизнес-вопроса:
1. Какой сегмент рынка приносит максимум выручки?
2. Как меняется прибыль по продуктам год к году?
3. Где себестоимость выше всего?

## Решение
- Загрузила данные в Power BI Desktop.
- Создала меры DAX: Total Sales, Total Profit, Total COGS.
- Построила три визуализации:
  - линейчатую диаграмму (выручка по сегментам)
  - группированные столбцы (прибыль по продуктам: 2013 vs 2014)
  - линейчатую диаграмму (себестоимость по странам)
- Настроила цвета, подписи данных, сортировку.

## Результаты
1. Выручка по сегментам: Government — 53 млн $, `Small Business` - 42 млн $.
2. Прибыль по продуктам: Paseo — лидер роста, Montana — аутсайдер.
3. Себестоимость по странам: USA и Canada — самые дорогие.

## Выводы для бизнеса
- Снизить зависимость от Government, развивая Enterprise и Midmarket.
- Инвестировать в Paseo, провести анализ Montana и Carretera (последняя очень хорошо взлетела за 2014 год).
- Пересмотреть ценообразование или поставщиков в USA и Canada.

## Инструменты
Power BI Desktop, DAX (SUM), визуальные настройки.
